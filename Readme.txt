
Provides java tools compatible with HEALPIX methods to work 
with sky pixelization.

AUTHORS:
            Nikolay Kuropatkin kuropat@fnal.gov
            
            
For detailed description see URL
                      http://home.fnal.gov/~kuropat/HEALPIX/PixTools.html
Or look in the doc subdirectory after installation of the package.
The Java doc of API is in the "doc" subdirectory too. 

Files in the repository:
  eag-HEALPix.jar - installation jar file.
  to install do : java -jar eag-HEALPix.jar
  
  eag-healpix-2012.07.28.src.tgz contains just source files of the package
